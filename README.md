# WORK TOGETHER

## FEATURES

- Live data editing
- Calculations (use =)
  - Example: = 5 + 20 // prints 25
- Cell referencing (use #{rNcM})

  - Example: = #{r1c1} // prints value in R1C1
  - Example: = #{r1c1} + #{R2C2} // prints sum of values in R1C1 and R2C2
